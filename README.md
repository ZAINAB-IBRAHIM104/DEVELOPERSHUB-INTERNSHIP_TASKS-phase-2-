# Global Superstore Dashboard 📊

A Streamlit dashboard for analyzing Global Superstore sales data.

## Features
- 📈 Interactive KPIs (Sales, Profit, Margin, Orders)
- 🔍 Dynamic filters ( Region, Category, Segment)
- 📊 Visualizations (Sales by Category, Regional Performance)
- 📊 Sales and Profit by Segment
- 📥 Download filtered data as CSV
